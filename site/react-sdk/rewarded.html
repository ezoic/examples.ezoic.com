React SDK

Rewarded ads with useEzoicRewarded

useEzoicRewarded() wraps window.ezRewardedAds with promise-based methods instead of raw callbacks, so a "watch a video for a reward" button is a normal async function.

What this demonstrates

The hook exposes request, show, requestAndShow, requestWithOverlay, and contentLocker, each resolving with a typed outcome object (status, reward, msg). It requires a loaderUrl pointing at the publisher's site-specific rewarded-ads loader script — the demo below leaves this blank by default (no publisher loader exists for examples.ezoic.com), so requests correctly stay pending until a real loader URL is supplied in the input field.

Hook usage
import { useEzoicRewarded } from '@ezoic/react-sdk';

function RewardButton() {
  const { requestAndShow } = useEzoicRewarded({
    loaderUrl: 'https://example.com/porpoiseant/ezadloadrewarded.js',
  });

  async function onClick() {
    const result = await requestAndShow({ placementId: 'unlock-article' });
    if (result.status === 'granted') {
      // unlock the reward
    }
  }

  return <button onClick={onClick}>Watch ad for reward</button>;
}
