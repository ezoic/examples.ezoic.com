Vue SDK

Rewarded ads composable

A visitor opts in to watch an ad in exchange for a reward — unlocked content, in-game currency, and so on. useEzoicRewarded() wraps window.ezRewardedAds in typed, promise-based calls.

What this demonstrates

Rewarded ads use a separate, publisher-specific loader script, so you enable them by passing its URL to EzoicPlugin (the rewardedLoaderUrl option) — find the exact host in your Ezoic dashboard. Every callback-style method (request, show, requestAndShow, requestWithOverlay, contentLocker) returns a Promise that settles for every outcome, including no-fill and cancellation, and a reactive status tracks the lifecycle (idle → initiated → displayed → closed).

Component usage
<script setup lang="ts">
import { useEzoicRewarded } from '@ezoic/vue-sdk';

const { requestAndShow, status, ready } = useEzoicRewarded();

async function unlock() {
  const r = await requestAndShow({ rewardName: 'premium_article' });
  if (r.reward) {
    // grant the reward — the visitor watched the ad
  }
}
</script>

<template>
  <button :disabled="!ready" @click="unlock">Watch an ad to unlock</button>
  <p>Rewarded status: {{ status }}</p>
</template>
